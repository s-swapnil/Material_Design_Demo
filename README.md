# Material_Design_Demo
A project showcasing the fundamental components of Material Design for Android.

Note-You can switch between activities using the Raised button in MainActivity class

Components showcased by activity-

MainActivity1:
-SwipeRefreshLayout
-Custom Toolbar
-Navigation Drawer
-AppCompatEditText
-Flat and Raised Buttons
-FLoating Action Button
-AppCompatCheckedBox
-AppCompatRadioButton
-SwitchCompat
-Circular Progressbar
-Linear Progressbar
-Alert Dialog
-Confirmation Dialog
-Progress Dialog

MainActivity2:
-SnackBar
-Bottom Navigation
-ViewPager(for image slide show)
-Tab Layout

MainActivity3:

-Collapsing Toolbar Layout
-CardView
-RecyclerView

MainActivity4:

-AppCompatSeekbar
-MaterialRangebar
-Date picker
-Time picker
